# Personal website outline

Working outline for `venkatnadella.com`, with `venkatnadella.in` redirected to the `.com`.

The site should stay simple, clean, and name-led. It is a minimalist academic-style homepage / proof-of-work archive, not a second publication and not the public home of Yukti.

## Core decision

**Site name:** Venkat Nadella

**Canonical domain:** `venkatnadella.com`

**Redirect:** `venkatnadella.in` -> `venkatnadella.com`

**Role:** personal website for papers, datasets, teaching materials, selected work, contact, and light personal information.

**Boundary:** Yukti / Read Yukti remains the publication. The personal site links to it prominently but does not absorb its identity.

**June 10 proof-of-work package:** EPPAI lecture notes, the AI Governance Primers, and the governance stress-test methodology note.

**Deferred:** datasets, ORCID / Google Scholar links, custom slide decks, and sample governance stress-test work.

## June 10 minimum viable version

Ship only what is needed:

1. Homepage.
2. Writing page with Yukti link.
3. Research / selected work page.
4. Teaching page.
5. Contact page.

Notes, datasets, expanded project pages, and personal/fun sections can come later.
